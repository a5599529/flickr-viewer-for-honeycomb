Flickr Viewer version 1.0.5 - What's new

  • Add recent tag search suggestion support;
  • Refine the recent activity process a little to show more items when clicking the main menu;
  • Add setting option to define the tag search mode, 'AND' or 'OR';
  • Bug fix: Mark a photo as favorite before authentication will fail, fixed;

Flickr Viewer version 1.0.4 - What's new

  • Refine the notification process;
  • Refine the fling gesture process on the photo detail view;
  • Click the avatar of the photo owner in the detail page to view his flickr web page in the browser so you can easily add him/her as your contact, since the flickr api seems not have a way to add someone as your contact;
  • Add section support in photo pool list of the photo detail screen;
  • Add tag search view on the action bar, so you can search photos based on tags. Use 'space' to separate the tags;
  • Add setting option to show/hide the application title, since some page has a rather long title, so the search widget has no room to display;

Flickr Viewer version 1.0.3 - What's new

  • Move the fling gesture to the list views, that is, the EXIF, comment and photo pool list views, to make it easier to switch among them;
  • Add 3 radio buttons to switch among EXIF, comment and photo pool views to make the UI more intuitive;

Flickr Viewer version 1.0.2 - What's new

  • Fix the 'force close' bug when network is not available;
  • Remove the menu item to switch between EXIF and comments view, instead, fling support added on the title bars of EXIF and comment list view;
  • Add photo pool information for each photo;
  • Click a photo pool list item of the photo detail screen to view all the public photos of the pool;
  • Double tap on the photo in the detail screen to view the big photo;
  • Make links in photo description clickable;

Flickr Viewer version 1.0.1 - What's new

  • Show messages when saving a photo, and the feedback when a photo was set as the wallpaper or saved;
  • Only show contacts who have photos uploaded when contact screen comes from the notification;
  • Add another filter in contact screen to filter contacts who have new photos uploaded;
  • Scroll to the first position of the photo grid after pagination;
  • Hide the menu item 'Show Owner's Photo' from the photo detail screen when viewing my own photo;
  • Add settings for notifications on my contacts upload photos and activities on my own photos;

Flickr Viewer version 1.0 key features:

  • Show interesting photos, my own public photos, my favorite photos, etc;
  • EXIF information shows in the photo detail screen, so you can know how the wonderful photos were taken;
  • View big photo, save it locally, set it as the wallpaper;
  • Share wonderful photos to other applications, like twitter, facebook, sina weibo, google plus, etc;
  • When sharing a photo to, say, twitter or g+, the photo URL is copied to clipboard, so you can just paste it to the text edit field in case the application to be shared does not accept extra text.
  • Context menu for favorite photos to remove one from the favorite list;
  • Show recent activities on your photos, and the photos you commented before;
  • Notifications when your contacts uploaded photos;
  • Notifications when your photos got comments or added as someone's favorite;
